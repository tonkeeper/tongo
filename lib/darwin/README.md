### Guide to build for macOS

#### Install lib

1) `brew tap ton-blockchain/ton`
2) `brew install ton`

#### Upgrade lib

1) brew update
2) brew reinstall ton

When you see the successful status of the build, you can find the `libemulator.dylib` file in the `/opt/homebrew/lib`
folder.
